# Frontend part for Cinema Platform // SoftServe Practice

Check [cinema-platform-back](https://github.com/stkossman/cinema-platform-back) for backend part.

## Tech Stack
- **Core:** React + TypeScript + Vite
- **Styling:** Tailwind CSS v4
- **Linting/Formatting:** Biome.js
- **Forms:** React Hook Form + Zod
- **Icons:** Lucide React
